02 · 四个核心方向

围绕模型生态、工具链、芯片定义、前沿研究形成闭环

四个方向不是孤立任务，而是一条持续闭环。客户交付不是第五个独立方向，而是它们的共同价值出口。

DIRECTION 01

量化、导出与评估工具链

如何把模型交付从专家手工经验，变成标准化、自动化、可复现的工程流程？

覆盖模型导入 → 图优化 → 量化配置 → HMONNX 导出 → Golden 生成 → 精度评估 → 性能评估 → CI 回归 → ModelZoo 发版 → 客户交付包的完整链路。把一次模型适配经验沉淀成下一次可复用的工具能力。

XH2Quantool GPTQModel HMONNX ModelZoo HMOps CudaGraph
DIRECTION 02

模型量化与芯片适配

如何让前沿模型真正变成后摩芯片上的可运行、可评测、可交付能力？

覆盖模型结构分析 → 算子识别 → 图改写 → 量化策略 → HMONNX/Golden 导出 → 精度评测 → 编译器联调 → 板端验证 → 客户复现的完整过程。把外部模型生态转化为公司芯片生态的一部分。

Qwen3 / 3.5 / 3.6 Qwen3-VL Gemma4 GLM / DeepSeekOCR ASR / TTS VLA / World Model
DIRECTION 03

下一代芯片算法定义

未来芯片应该支持什么，才能更好地承载下一代 AI 模型？

从模型和算法侧输入关键需求：哪些数据类型值得支持、哪些算子需要硬件友好、哪些大算子需要融合、哪些访存和 cache 瓶颈需要在架构上解决。用真实模型和真实算法瓶颈校准未来芯片方向。

月辉 / YH XH3 / M50 Reference Model PIM + DDR FP4 / MXFP / VQ Hadamard / TurboQuant
DIRECTION 04

前沿算法跟进与研究

AI 模型和算法变化这么快，公司应该提前理解什么、判断什么、储备什么？

既包括前沿模型跟进，也包括先进算法研究：跟踪模型趋势，研究量化算法、推理优化、压缩方向、低比特格式，产出论文、专利和可工程化算法储备。成为公司内部的模型趋势雷达和算法武器库。
